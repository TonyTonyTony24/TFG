## 📌 Descripción
**AutoGestor** es una plataforma web diseñada para ayudar a conductores y empresas a llevar un control completo sobre sus vehículos. 
Permite gestionar mantenimientos, pagos, alertas, repuestos y más, todo en un solo lugar. Ideal tanto para usuarios individuales como para flotas empresariales (logística, reparto, etc.).


## 🎯 Objetivos principales
- Registro de mantenimientos (cambios de aceite, revisiones, etc.)
- Gestión de varios vehículos por usuario
- Alertas para ITV, seguros, impuestos, etc.
- Acceso al despiece del vehículo por matrícula o modelo
- Sugerencias de repuestos y enlaces a tiendas
- Historial completo de mantenimientos, km y gastos.

  
## 🛠 Tecnologías utilizadas
- Lenguajes: HTML5, CSS3, JavaScript, PHP
- Frameworks:
  - **Frontend**: Angular
  - **Backend**: Symfony (o PHP puro)
- Base de datos: MySQL
- Otras herramientas:
  - Git y GitHub para control de versiones
  - Figma para diseño de interfaces (opcional)
  - Docker (opcional, si lo usas para despliegue local)
